How to add your own Tiles

Each tile should have it's own directory under resources/tiles.

Tile file

First of all a tile file should be created, as described in the Tile File section.

Textures

After creating the tile file, all textures specified in the file (no path should be specified) are to be added to a 'textures' directory within the tile directory.

Using the Tile

Once the tile is created, it can automatically be used in the map editor.

Example

An example tile can already be found in the users resource directory.
